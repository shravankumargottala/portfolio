Code Assessment Tool

Project Description:


Code Assessment Tool is designed to help the developer/reviewer to check if the standard coding formats are being followed in Informatica, UNIX and Teradata (BTEQ) scripts, It points out the exact line of code at which the violation is identified and defines whether the identified issue is an error or a warning and states the coding standard that has been violated. It enables developer to have a quick reference at the code and make necessary changes so as to promote the code to next higher environment (SIT, UAT, IR, PROD). Automated project reduce the possibility of errors by the Application Development teams in the Source Codes and also minimize the efforts of the Code Reviewers, by sending the consolidated report to the developer and/or reviewer.


Responsibilities:


  • Understanding and analyzing the requirements by interacting with the application teams.

  • Participating in implementation/release meetings of new standards and analyzes the impacting applications and plan the applications to handle the risk based on the impact.

  • Helping out the team in resolving and improving the script performance and ensuring that the project teams are following the coding standards.

  • Defining the logic to be performed for the standards and coding the standards using Python and Teradata BTEQs.

  • Designing test cases and performing testing in all possible ways in all lower environments like system testing, unit testing and regression testing.

  • Preparing the test case documents for each test case and documenting the test results.

  • Resolving the production issues and providing support to application users.
